Do some tests!

Input the number of test cases as an integer, optionally in "E-notation". E.g., a million can be represented as "1000000" or "1e6". Plz add 0's before decimal points when necessary, i.e., ".13e6" is not valid, but "0.13e6" is. Non-integers will be rounded up.
